# Saturn.fm
![](saturn-fm.png)

A recreation of the Sega Saturn CD Player UI in Javascript using three.js, React & Redux.

## Getting Started

1. `npm install` to install dependencies
2. `npm start` to start webpack-dev-server
3. Start jammin' out at `localhost:3000`
